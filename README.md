# 📓 Jupyter Notes

¡Bienvenido a mi colección de cuadernos interactivos! 🚀 Este repositorio es mi espacio personal de documentación y aprendizaje. Aquí traduzco conceptos teóricos y técnicos en apuntes claros, directos y, lo más importante, **con código 100% ejecutable** dentro de Jupyter Notebooks.

## 📂 Contenido del Repositorio

Actualmente, los apuntes se dividen en las siguientes áreas de estudio:

### 🐹 1. Básicos de Go (Golang)
Exploración de la simplicidad y el poder de concurrencia de Go.
* **Sintaxis y Tipos:** Declaraciones cortas, inferencia de tipos y colecciones dinámicas (`slices` y `maps`).
* **Estructuras de Control:** El uso del todopoderoso `for` y la elegancia del `switch`.
* **Concurrencia:** Introducción práctica a las *Goroutines* y cómo Go ejecuta múltiples tareas de forma simultánea y ligera.

### 🦀 2. Básicos de Rust
Documentación sobre el lenguaje de sistemas que prioriza la seguridad y el rendimiento extremo.
* **El Sistema de Tipos:** Desde primitivos estrictos (`i32`, `u8`, `char`) hasta `Structs` y `Enums` avanzados.
* **Control de Flujo:** Manejo seguro de ciclos y el uso del poderoso (y exhaustivo) condicional `match`.
* **Seguridad sin GC:** Cómo escribir código predecible y evitar fallos de memoria.
* **Ausencia de Null:** Uso de los tipos `Option<T>` y `Result<T, E>` para un manejo de errores robusto.

### 🧠 3. Arquitectura de Von Neumann
Un repaso teórico al modelo conceptual que da vida al hardware moderno.
* Análisis del diseño clásico que define a casi todas las computadoras actuales.
* Explicación de los componentes críticos: Unidad Central de Procesamiento (ALU y Control), Memoria y dispositivos de Entrada/Salida.

## 🛠️ Entorno y Requisitos

Si deseas clonar este repositorio y ejecutar las celdas de código en tu propia máquina (ideal para entornos Linux), asegúrate de tener instalados los siguientes *kernels* en tu entorno de Jupyter:

* **Para Go:** Necesitarás instalar el kernel [gonb](https://github.com/janpfeifer/gonb).
* **Para Rust:** Necesitarás instalar el kernel [evcxr_jupyter](https://github.com/evcxr/evcxr/tree/main/evcxr_jupyter).
